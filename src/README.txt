Dotted style theme for Sphinx.

Features
========
* provide ``dotted`` theme for render HTML document.


Set up
======
Make environment with pip::

    $ pip sphinxjp.themes.dotted


Convert Usage
=============
setup conf.py with::

    html_theme = 'dotted'

and run::

    $ make html


Requirement
===========
* Python 2.7 or later (not support 3.x)
* Sphinx 1.2.x or later.
