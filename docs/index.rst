=======================================
 Notebok style theme sample for Sphinx
=======================================

Quick start
===========

install:

.. code-block:: bash

    $ easy_install sphinxjp.themes.notebook


setup your ``conf.py`` with:

.. code-block:: bash

    extensions = ['sphinxjp.themescore']
    html_theme = 'notebook'

and run:

.. code-block:: bash

    $ make html

then your will get this page's style HTML output.
