======================================
 Dotted style theme sample for Sphinx
======================================

Quick start
===========

install:

.. code-block:: bash

    $ easy_install sphinxjp.themes.dotted


setup your ``conf.py`` with:

.. code-block:: bash

    extensions = ['sphinxjp.themescore']
    html_theme = 'dotted'

and run:

.. code-block:: bash

    $ make html

then your will get this page's style HTML output.
