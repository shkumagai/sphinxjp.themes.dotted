HTML theme options
==================

:slidetoc:
   Enable global toc (table of contents) tree on top-navigation bar. Defaults to false.

:enablesidebar:
   Enable to show the sidebar on the left side of document. Defaults to false.

:rightsidebar:
   Put the sidebar on the right side. Defaults to false.
